Introduction

Welcome to the Klavi development documentation and API reference!

Klavi payment initiation product support two method integration: Payment page and Payment API.

  • Payment page: The simplest and fastest integration method for payment initiation.
  • Payment API: This approach requires partners to have back-end system development and integration capabilities.
note

Before you get started, you must first create a Klavi account and create an API accessKey and secretKey. If you have any questions along the way, please don’t hesitate to talk to us at suporte@klavi.ai.
